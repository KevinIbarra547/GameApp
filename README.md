# Fortress Fall
A 2D building and defense game inspired by Fortnite, built with PHP and JavaScript.

## How to Play
- Start at the Login page.
- Use the Lobby to equip your power-ups.
- Build your base in the Build Phase (30 seconds).
- Defend against the boss in the Defend Phase!
